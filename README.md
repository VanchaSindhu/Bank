# Django-chat-app
- A Python project for Bank application created using Django HTML content, and more 

 
Features
- Creating Account.
- Generating Pin and Encripting the pin .
- Displaying Balance.
-depositing Amount in an account.
- Withdrawing amount from an account.
- Account Transfer.
